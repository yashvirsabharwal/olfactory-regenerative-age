# Changelog

## 2026-06-25

- M3.1: Logged that the GSE184117 author label request was sent to Bradley Goldstein on 2026-06-25 and that the study team responded the same day. Raw sequencing files and 10x matrices are public through GEO, sample metadata can be matched through GEO record names plus manuscript Table 1, and no separate per-cell manual annotation table is available.
- M4.1: Added `docs/foundation_model_benchmark_plan.md`, covering Geneformer, scGPT, scFoundation, ORA, scVI, hybrid, and aging-clock-style benchmark scope, donor-level evaluation rules, failure criteria, and claim language.
- M4.2: Added `make foundation-benchmark-subsets`, generated fixed Gateway benchmark H5ADs for 120k lineage cells, 180k epithelial cells, and 250k all cells, and exported ORA donor folds plus Gateway gene symbol/ID manifests for model tokenizers.
- M4.4: Added `make foundation-model-benchmark` and ran local Geneformer V1-10M encoder-only embeddings on 24k lineage cells across 202 donors. Geneformer covered 98.96% of Gateway genes, ran on MPS in 1,442 seconds with 3.16 GB peak RSS, and produced a best repeated-CV donor age MAE of 15.04 years; scGPT and scFoundation are transparently deferred pending checkpoint/weight staging.
- M4.5: Added a leakage-safe aging-clock-style expression baseline with fold-internal gene variance selection, scaling, and PCA on donor-level genomewide pseudobulk logCPM. The best strict expression-clock model is ridge at MAE 11.98 years, improving over ORA+scVI but not reaching the <10-year target.
- M5.1: Added `make regeneration-axis-feature-map`, a controlled regeneration-axis feature map for all 120 ORA/module features, literature-anchored expected age directions, Gateway age/importance joins, cross-tissue specificity joins, a theme summary table, and manuscript figure outputs.
- M5.2: Added 14 curated OE regeneration pathway modules with source/citation metadata, full Gateway module scoring, coverage checks, primary/strict unadjusted and covariate-adjusted age associations, and module-to-ORA correlation tables. Primary adjusted associations significantly decline with age for GAP43/DCX immature-OSN and OMP/ADCY3 mature-OSN modules; strict-cohort adjusted associations are broader and context-sensitive.
- M5.3: Added a first-pass regulatory-driver hypothesis layer from curated TF/pathway target programs scored on genomewide pseudobulk logCPM by donor and lineage group. The local fallback ranks WNT/CTNNB1, ASCL1, STAT/IRF, TP63, OMP/ADCY3, NOTCH/HES, YAP/TEAD, and NF-kB/IL17/TNF as leading driver hypotheses while deferring decoupler/SCENIC-style regulon inference.
- M5.4: Added `make niche-signaling-analysis`, curated LR priors, donor-level sender-receiver pseudobulk scoring, age/ORA association tables, a niche-driver priority table, and an extended-data niche figure. Top hypotheses include immune-to-lineage IFN/TNF/chemokine/MIF signals, often declining with age/ORAA, so they remain context-sensitive niche hypotheses rather than mechanism proof.
- M6.1/M6.2/M6.3: Added `make cross-tissue-specificity` and `make cross-tissue-age-effects`, a cross-tissue comparator plan/candidate matrix, first-pass classification of all 120 ORA/module features, and donor-level CELLxGENE nasal/bronchial comparator age-effect estimates. The feature classification separates 31 olfactory-specific, 26 airway/nasal shared, 12 pan-epithelial regenerative, 49 immune/inflammatory shared, and 2 not-comparable features; the age-effect stage generates 71 adult nasal/bronchial estimates, with 3 nominal p<0.05 and 0 within-dataset/scope FDR<0.05 effects.
- M6.4: Added `make spatial-validation-plan`, a public spatial-data triage, 9-panel ORA histology/spatial marker design, donor-level readout plan, and `docs/spatial_perturbation_validation_plan.md`. No direct public adult human olfactory epithelial spatial aging dataset was found; GSE235714, GSE292993, and GSE303809 are recorded as context-only spatial resources.
- M6.5: Added `make perturbation-validation-plan`, a public organoid/ALI perturbation triage, minimum adult-OE perturbation experiment design, and `docs/perturbation_validation_plan.md`. No clean adult human olfactory aging perturbation dataset was found; GSE309325 and GSE299529 are the top adapter candidates.
- M7.3: Added `make regeneration-dynamics`, an H5AD fate-method input audit, and a Scanpy diffusion-pseudotime pilot on 40k basal-to-neuronal lineage cells. Velocity/CellRank fate inference is blocked by missing spliced/unspliced layers; DPT gives exploratory topology support only (Spearman r = 0.598), not lineage flux.

- M0.5: Added explicit allowed/prohibited claim language and a manuscript claim-audit checklist.
- M1.1: Installed a local OpenMP runtime for native XGBoost/LightGBM loading and reran the core ORA feature, module, benchmark, permutation-null, interpretation, and model-card artifacts from the corrected M0 code.
- M1.2: Added donor-level CLR compositional age modeling with primary, strict-threshold, and single chemistry/collection sensitivity scenarios; generated compositional summary/sensitivity tables and an extended-data figure.

- M4.6: Added an ORA+scVI hybrid benchmark that merges composition/module features with donor scVI embeddings, reruns matched repeated-CV candidate models, and summarizes feature-family importance. The hybrid histogram-gradient model is now the best benchmark at MAE 12.90 years.
