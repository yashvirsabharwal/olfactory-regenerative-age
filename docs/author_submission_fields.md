# Author And Administrative Submission Fields

Updated: 2026-06-23

Purpose: collect the non-analysis fields required by most journals before final submission. These require author confirmation and should not be inferred from the analysis repository.

| Field | Current status | Required content |
| --- | --- | --- |
| Final author list | pending | Ordered names, degrees if journal requires, affiliations, ORCID IDs if available. |
| Corresponding author | pending | Name, email, mailing address if required. |
| Author contributions | pending | CRediT roles or target-journal contribution format. |
| Funding | pending | Grant numbers, institutional support, or "no specific funding" statement. |
| Competing interests | pending | Author-confirmed conflict statement. |
| Ethics/consent | pending | Source-study human subjects / consent statement for Gateway and public GEO reuse; cite source study if appropriate. |
| Acknowledgements | pending | Data providers, computational resources, colleagues, and non-author contributions. |
| Suggested reviewers | optional pending | Names, institutions, emails, expertise, conflicts checked. |
| Opposed reviewers | optional pending | Names and concise rationale if journal permits. |
| Preprint policy | pending | Whether to post preprint before submission and where. |
